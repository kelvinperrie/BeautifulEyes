Have to have an element that we can put the eyes in; in this case the td with id of example4 
<td id="example4">
    <img src="images/stock-potato.jpg" alt="potato"  />
</td>
NOTE: the container where you are inserting the eyes must be able to contain absolute elements (e.g. give it a relative position) 
#example4 {
    position: relative;
}
Create the eyes as needed 
$('#example4').makeEye({ size: 24, position: { x: 78, y:82 }, eyeBallSize: 8 });
$('#example4').makeEye({ size: 24, position: { x: 118, y:82 }, eyeBallSize: 8 });
size = how big the eye is (in px)
position = where the eye will be centered
eyeBallSize = the size of the eye ball (default is 4)
Passing only the defaults 
$('#example5').makeEye({ size: 18, position: { x: 138, y:92 } });
$('#example5').makeEye({ size: 18, position: { x: 168, y:92 } });
Making some BIG eyes 
$('#example6').makeEye({ 
    size: 30, 
    position: { x: 150, y:135 }, 
    eyeBallSize: 15, 
    borderWidth: 4 
});
$('#example6').makeEye({ 
    size: 30, 
    position: { x: 207, y:135 }, 
    eyeBallSize: 15, 
    borderWidth: 4 
});
borderWidth = the width of the line around the outside of the eyes
